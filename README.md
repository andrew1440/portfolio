# Andrew Odongo — Nairobi's Digital Explorer

A premium, minimalist portfolio redesign inspired by the energy of Nairobi’s "Silicon Savannah" and modern high-end developer aesthetics. This project focuses on high-impact typography, glass-morphism, and advanced motion design.

## 🌃 The Theme: Nairobi After Dark
The design uses a deep, rich palette (Nairobi After Dark) with matatu-orange accents, reflecting a blend of grassroots community work and high-tech AI exploration.

## 🚀 Key Features
- **Modern Minimalist UI**: Clean, high-contrast layouts using the **Syne** and **Inter** font families.
- **Advanced Motion**: Leverages **GSAP** and **ScrollTrigger** for smooth section reveals, magnetic buttons, and a custom interactive cursor.
- **Interactive "Pulse"**: A marquee and card-based section highlighting the culture and values behind the code.
- **Bento-style Projects**: A clean grid for showcasing impact-driven projects with hover-reveal imagery.
- **Dynamic Background**: Animated gradient orbs that move subtly in the background to create depth.
- **Fully Responsive**: Crafted to look premium on everything from mobile screens to 4K monitors.

## 🛠️ Technical Stack
- **HTML5 & CSS3** (Vanilla with custom design system)
- **JavaScript** (Modern ES6+)
- **GSAP** (GreenSock Animation Platform)
- **ScrollTrigger** (Advanced scroll-driven interactions)
- **Syne & Playfair Display** (Typography)

## 📁 Project Structure
- `index.html`: Main structure and content blocks.
- `style.css`: Custom design tokens, glass-morphic utilities, and layout.
- `app.js`: GSAP logic, cursor interactions, and scroll animations.
